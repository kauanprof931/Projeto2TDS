# Commitando minhas atualizações
Para exportar seu projeto da máquina até seu repositório GitHub,
basta realizar as seguintes operações:

git add *
Para selecionar todo o projeto

git commit -m "mensagem"
Para descrever sobre o que é aquela atualização

git push origin main
Para realizar o envio

Essas etapas devem ser seguidas em ordem, também não podemos
ignorar nenhum comando, pois isso pode fazer nosso commit não
ser enviado.


Atribui a função diretamente a variável
const somar = function(a, b) {
    return a + b;
};

const saudacao = function(nome) {
    return `Olá, ${nome}!`;
};

Método moderno
const saudacao = (nome) => `Olá, ${nome}!`;
